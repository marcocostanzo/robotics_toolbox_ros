TooN::Vector<3> r2eul ( const TooN::Matrix< 3, 3 > &  R)

Convert rotation matrix to Euler angles.

Compute the ZYZ Euler angles (1x3) corresponding to the rotation matrix R (3x3). The 3 angles EUL=[PHI,THETA,PSI] correspond to sequential rotations about the Z, Y and Z axes respectively.

Notes::

  • AXES ZYZ
  • There is a singularity for the case where THETA=0 in which case PHI is arbitrarily set to zero and PSI is the sum (PHI+PSI).
  • Translation component is ignored.
Parameters
Ra rotation matrix
Returns
a vector containing [PHI,THETA,PSI]
